# Lineage Foundation Website

Source for [lineage.foundation](https://lineage.foundation).

## Development

Static HTML/CSS/JS. Edit files and push to `main` — Vercel deploys automatically.

## Layout

- `index.html` – Landing page
- `style.css` – Styles
- `script.js` – Client-side scripts

## Deployment

Deployed on [Vercel](https://vercel.com). See [lineage-foundation/platform/docs/website-setup.md](https://github.com/lineage-foundation/platform/blob/main/docs/website-setup.md) for setup.
